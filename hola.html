¡Hola! Vamos a resolver el sistema de ecuaciones paso a paso:

[ \begin{cases} 2x + y - z = -5 \quad \text{(1)} \ 4x + 5y - 3z = 8 \quad \text{(2)} \ 2x + 7y - 2z = -3 \quad \text{(3)} \end{cases} ]

Paso 1: Eliminar una variable

Observamos que las ecuaciones (1) y (3) tienen el mismo coeficiente para ( x ) (ambas son ( 2x )). Restemos la ecuación (1) de la ecuación (3):

[ (2x + 7y - 2z) - (2x + y - z) = -3 - (-5) ]

Simplificando:

[ 6y - z = 2 \quad \text{(4)} ]

Paso 2: Eliminar la misma variable usando otra pareja de ecuaciones

Multipliquemos la ecuación (1) por 2 para igualar los coeficientes de ( x ) en las ecuaciones (1) y (2):

[ 2 \times (2x + y - z) = 2 \times (-5) \implies 4x + 2y - 2z = -10 \quad \text{(1a)} ]

Restemos la ecuación (1a) de la ecuación (2):

[ (4x + 5y - 3z) - (4x + 2y - 2z) = 8 - (-10) ]

Simplificando:

[ 3y - z = 18 \quad \text{(5)} ]

Paso 3: Resolver el sistema de dos ecuaciones obtenidas

Ahora tenemos:

[ \begin{cases} 6y - z = 2 \quad \text{(4)} \ 3y - z = 18 \quad \text{(5)} \end{cases} ]

Restemos la ecuación (5) de la ecuación (4):

[ (6y - z) - (3y - z) = 2 - 18 \implies 3y = -16 \implies y = -\frac{16}{3} ]

Paso 4: Encontrar ( z ) usando el valor de ( y )

Sustituyamos ( y = -\frac{16}{3} ) en la ecuación (5):

[ 3\left(-\frac{16}{3}\right) - z = 18 \implies -16 - z = 18 \implies -z = 34 \implies z = -34 ]

Paso 5: Encontrar ( x ) usando los valores de ( y ) y ( z )

Sustituyamos ( y = -\frac{16}{3} ) y ( z = -34 ) en la ecuación (1):

[ 2x + \left(-\frac{16}{3}\right) - (-34) = -5 ]

Simplificando:

[ 2x - \frac{16}{3} + 34 = -5 \implies 2x + \frac{86}{3} = -5 ]

Restamos (\frac{86}{3}) de ambos lados:

[ 2x = -5 - \frac{86}{3} = -\frac{15}{3} - \frac{86}{3} = -\frac{101}{3} ]

Finalmente:

[ x = -\frac{101}{6} ]

Solución Final:

[ \boxed{ x = -\frac{101}{6}, \quad y = -\frac{16}{3}, \quad z = -34 } ]
